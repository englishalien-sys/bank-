Start
The process begins when the user inserts the ATM card
Language Selection
The ATM asks the user to select a language.
Options include:
English
International
Domestic
Mobile Number & OTP (for first-time / PIN setup)
The system checks the mobile number linked to the account.
An OTP is sent to the registered number.
The user enters the OTP.
The user sets a new PIN (number between 10–99 as shown).
The user confirms the PIN.
If confirmed correctly → Welcome screen is shown.
Enter PIN
The user is prompted to enter the PIN.
The system checks:
Correct PIN → Proceed
Incorrect PIN → Ask user to re-enter the PIN
Main Banking Options
After successful PIN verification, the user can choose:
Bank Balance
Withdraw
Deposit
Other Banking services
Bank Balance
If selected, the ATM shows the account balance.
Then proceeds to take receipt / eject card.
Withdraw Cash
User enters the withdrawal amount.
User re-enters the PIN for confirmation.
If PIN is correct:
ATM dispenses cash → “Please take your amount
If PIN is incorrect:
User is asked to re-enter the PIN
Receipt is printed.
Deposit
User inserts cash notes.
ATM counts and displays the total number of notes.
User enters PIN for confirmation.
Deposit is completed.
Receipt is printed.
End Process
ATM displays:
“Take your receipt”
“Please take the card”
Transaction ends.

This diagram represents a complete ATM workflow, including:
Card insertion

PIN validation
